# 一句话结论

《人类简史》的核心价值，是帮助读者把个人处境放回更长的文明尺度中重新观察。

# 这本书在解决什么问题

- 人类为何能在群体协作上远超其他物种
- 宗教、货币、国家与公司为什么能稳定存在
- 现代社会的进步是否必然等于更幸福

# 核心观点

## 认知革命

智人的优势不只是工具能力，而是能够相信和传播共同想象，从而组织大规模协作。

## 想象秩序

国家、货币、公司等制度之所以稳定，不是因为天然存在，而是因为大量人持续共同相信。

## 科学革命

现代社会的跃迁来自承认无知，并围绕未知持续投入研究、资本和组织能力。

# 认知模型

- 文明尺度观察：从个体问题切换到制度和时代背景
- 共同想象分析：识别一个系统靠什么叙事维持运转
- 进步悖论：区分效率提升与幸福提升

# 我的理解

这本书特别适合作为“世界观重置”读物。它不直接给行动建议，但能显著提升你理解制度、组织和叙事力量的能力。

# 实践应用

- 在分析行业或组织时，同时拆解资源结构和叙事结构
- 遇到看似稳定的规则时，反问它依赖哪些集体共识
- 建立跨历史尺度的思考习惯，减少短期情绪绑架
