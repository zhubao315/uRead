《道教与养生》

  • 作者: 陈撄宁
  • 标签: 读书笔记 哲学宗教 仙学 道教养生 静功实修 女子丹法 得大自在
  • 评分: ⭐⭐⭐⭐⭐
  • 状态: 待读
  • 来源: 道教与国学养生书单 得大自在精选书单

👨‍🏫 作者介绍：陈撄宁 (Chen Yingning)

陈撄宁（1880—1969），号“圆顿子”，安徽怀宁人。中国近代道教协会第二任会长，被誉为**“当代的太上老君”**。

  • 生平：早年习儒，后因身体孱弱（患肺痨）弃文从医，进而修道。遍访名山大川，博览《道藏》，终成一代宗师。他一生未出家，以居士身份弘扬道学。
  • 成就：创立了现代**“仙学”**（Xianology）体系，主张将道教的修炼技术从宗教迷信中剥离出来，作为一门独立的生命科学来研究。创办《扬善半月刊》、《仙道月报》，保存了大量珍贵的丹道资料。

📅 创作背景

  • 时代背景：民国时期，国运衰微，西学东渐。传统道教被视为迷信糟粕，且道教内部丹法失传，多流于斋醮科仪。
  • 写作动机：陈撄宁痛感“道教虽衰，而神仙之学不可废”。他试图用科学和逻辑的语言，重新阐释古老的丹道，提倡**“科学的神仙家”**，强调实证与健康。

📑 书籍大纲

本书是陈撄宁先生的文章汇编，涵盖了他一生的学术精华：

  1. 仙学总论：界定“仙学”的独立地位，区别于儒、释、道三教。
  2. 静功问答：极其珍贵的实修问答录，解答了初学者关于打坐、呼吸、意念的无数困惑。
  3. 典籍注释：对《黄庭经》、《孙不二女丹诗》等经典的深度注解。
  4. 女子丹法：陈撄宁是近代公开传授女子内丹术的第一人，专章论述女性生理特点与修炼方法。
  5. 辨伪存真：批判当时流行的各种旁门左道。

📝 主要内容与核心观点

1. 仙学的定义

陈撄宁提出：“仙学的主旨在长生久视，不是死后升天。”他认为神仙不是鬼神，而是进化了的人类。仙学是一门通过修炼改变生理结构，从而延长寿命的**“生命科学”**。

2. 静功：听皮肤呼吸

这是陈撄宁最著名的实修法门。

  • 方法：在静坐时，不要守丹田，不要数呼吸，而是将注意力放在**“听皮肤毛孔的呼吸”**上。
  • 原理：通过微细的体表感知，让大脑皮层进入深度抑制状态，从而启动身体的先天呼吸（胎息）。这比传统的“意守丹田”更安全，不易走火入魔。

3. 三教圆融，独标仙学

他认为儒家讲做人（世间法），佛家讲修性（出世法），而道家仙学讲**“性命双修”**，既能在世间建立功业，又能超脱生死的局限，是最圆满的。

4. 只有科学能解释仙学

陈撄宁极具前瞻性地指出，只有未来的科学（物理学、生物学）发展到一定高度，才能完全解释内丹术的奥秘。他鼓励弟子学习现代医学，用科学验证丹道。

☯️ 专项深入研究：养生实修与去魅

陈撄宁对房中术持审慎态度，但对“生理转化”有深刻见解。

1. 房中术的批判与吸收

陈撄宁认为，古代房中术多为“泥水丹法”，极易流于淫欲。

  • 观点：他主张**“清修为主”**，认为真正的能量（精）可以通过静功（听息）自然生发，不一定非要通过男女交感。
  • 例外：但他不否认对于老年亏损者，“彼家丹法”可能是一种有效的补救手段（栽接），前提是必须“无欲”。

2. 仙学对“漏丹”的看法

针对年轻人的遗精问题，陈撄宁给出了极具操作性的建议：

  • 睡姿：侧卧如弓，压住耳窍（肾开窍于耳）。
  • 心理：彻底斩断意淫。他强调“念头”是漏丹的根源。
  • 锻炼：推荐“冷水浴”和“静坐”来增强神经系统的固摄能力。

3. 女子丹法的开拓

他专门整理了适合女性的修炼体系。指出女性修炼重在**“斩赤龙”**（断经），方法与男性不同，先炼形（按摩乳房、腹部）后炼气，以血化气。这体现了道教在性别上的平等与差异化教学。

💬 金句摘录与解读

  1. “神仙是人做，不是神赐。”

    • 解读：打破了宗教的神秘主义。修仙不需要上帝的恩赐，不需要偶像崇拜，全靠自己的努力（修炼）去转化生命形态。这是极致的人本主义。

  2. “学仙必从静功入手。”

    • 解读：指出了路径。无论什么门派，如果不能静下来（入定），一切气脉、光影都是幻觉。静是生发万有的土壤。

  3. “与其求生于死后，不如求生于生前。”

    • 解读：对比佛教追求“死后往生极乐”，仙学追求的是“当下的肉体永生”或“长寿健康”。这更符合现代人对生活质量的追求。

🚀 应用指导：生活、学习与工作

1. 生活应用：听皮肤呼吸法（解压神器）

  • 原理：转移注意力，启动副交感神经。
  • 场景：失眠、焦虑、心烦意乱。
  • 操作：
    • 姿势：平躺或静坐。
    • 意念：闭眼，想象全身毛孔都在随呼吸开合。吸气时毛孔打开摄取能量，呼气时毛孔关闭锁住能量。
    • 效果：这能迅速将“上火”的大脑冷却下来，使气血遍布全身，手脚瞬间变暖。

2. 学习应用：建立“独立学科”意识

  • 原理：陈撄宁将仙学从宗教中独立出来。
  • 场景：面对跨界知识。
  • 操作：
    • 剥离外壳：学习任何传统文化（如中医、易经），尝试剥离其迷信、宗教的外壳，提取其**“技术内核”**（如中医的系统论、易经的二进制）。
    • 建立体系：像建立“仙学”一样，构建自己的知识体系，不盲从权威。

3. 工作应用：实证主义精神

  • 原理：不尚空谈，唯重实证。
  • 场景：方案验证。
  • 操作：
    • 亲身测试：陈撄宁为了验证丹法，曾多次烧炼外丹并亲自服食（甚至因此中毒）。在工作中，对于关键方案，必须进行A/B测试或灰度发布，用数据说话，而不是靠理论推导。
