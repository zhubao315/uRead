《商业模式新生代》（Business Model Generation）

亚历山大·奥斯特瓦德（Alexander Osterwalder） & 伊夫·皮尼厄（Yves Pigneur）

“商业模式就是描述一个组织如何创造价值、传递价值和获取价值的基本原理。“

作者介绍

亚历山大·奥斯特瓦德（Alexander Osterwalder）

生平背景 奥斯特瓦德是瑞士商业理论家、作家、顾问和企业家。他在洛桑大学获得政治学硕士学位，后获得管理信息系统博士学位。他的博士论文就是关于商业模式本体论的研究，这也成为了本书的理论基础。

主要成就

  • 商业模式画布之父： 他发明的”商业模式画布”（Business Model Canvas）已成为全球通用的商业设计工具，被数百万创业者和世界500强企业（如3M、爱立信、德勤）广泛使用。
  • Strategyzer创始人： 他创立了Strategyzer公司，致力于开发实用的商业创新工具。

影响力 奥斯特瓦德改变了商业计划书的撰写方式。过去，创业者需要写几十页的文档；现在，他们只需要在一张纸上画出9个模块。他让商业模式设计变得可视化、结构化和可协作。

创作背景

创作时间 本书于2010年出版。

社会环境 当时，传统的商业环境正在发生剧变，新行业不断涌现，旧行业面临重塑。企业家和管理者急需一种新的语言和工具来理解、设计和变革商业模式。

创作方式 这本书本身的创作过程就是一个创新的商业模式案例。它是由来自45个国家的470名”商业模式画布”实践者共同协作完成的。这种众包（Crowdsourcing）的创作方式保证了内容的实用性和普适性。

主要内容

1. 商业模式画布：9个构造块

这是全书的核心工具，用于在一张纸上描述任何商业模式。

  1. 客户细分（CS, Customer Segments）： 我们为谁创造价值？谁是我们最重要的客户？（大众市场、利基市场、多边市场等）
  2. 价值主张（VP, Value Propositions）： 我们向客户传递什么价值？我们帮助客户解决什么问题？（新颖、性能、定制、设计、价格、成本削减等）
  3. 渠道通路（CH, Channels）： 我们如何接触客户并传递价值？（直销、分销、网络、实体店等）
  4. 客户关系（CR, Customer Relationships）： 我们与客户建立什么样的关系？（个人助理、自助服务、社区、共同创造等）
  5. 收入来源（RS, Revenue Streams）： 客户愿意为什么付费？（资产销售、使用费、订阅费、租赁、授权、广告等）
  6. 核心资源（KR, Key Resources）： 运行该模式需要什么资源？（实体、知识、人力、金融等）
  7. 关键业务（KA, Key Activities）： 我们必须做什么才能让模式运作？（生产、解决问题、平台/网络等）
  8. 重要伙伴（KP, Key Partnerships）： 谁是我们的关键合作伙伴和供应商？（非竞争者联盟、竞合、合资、供应商关系等）
  9. 成本结构（CS, Cost Structure）： 所有的成本是多少？（固定成本、可变成本、规模经济、范围经济等）

2. 经典商业模式式样

书中总结了5种常见的商业模式原型：

  • 分拆商业模式： 将企业分为客户关系型、产品创新型和基础设施型三种业务，因为它们有不同的经济逻辑和文化，应该分拆管理。例如：私人银行、移动运营商。
  • 长尾模式： 销售少量、多样的产品。重点在于降低库存成本和强大的平台。例如：亚马逊、乐高工厂。
  • 多边平台模式： 将两个或更多独立但相互依存的客户群体连接在一起。价值在于网络效应。例如：Google（连接搜索用户和广告商）、Visa（连接商家和持卡人）。
  • 免费模式： 至少有一个客户群体可以享受免费服务。由另一个群体付费（广告模式）或通过增值服务付费（Freemium）。例如：百度、Skype、Flickr。
  • 开放式商业模式： 通过与外部伙伴合作来创造和获取价值（由外而内或由内而外）。例如：宝洁的”联系与发展”策略。

3. 设计思维在商业中的应用

商业模式设计不仅仅是填空，更需要设计思维。

  • 视觉化思考： 使用便利贴和白板，让思维可见，促进团队讨论。
  • 原型制作： 像产品设计师一样，快速制作多个商业模式原型（MVP），进行测试和迭代。不要爱上你的第一个想法。
  • 故事讲述： 用故事来描述新模式如何解决客户问题，让投资者和员工更容易理解。

4. 蓝海战略视角

结合《蓝海战略》的四步动作框架（剔除、减少、增加、创造）来创新商业模式。

  • 剔除： 行业中习以为常的哪些因素可以被剔除？（降低成本）
  • 减少： 哪些因素可以减少到行业标准以下？（降低成本）
  • 增加： 哪些因素应该增加到行业标准以上？（提升价值）
  • 创造： 行业中从未有过的哪些因素应该被创造？（创造新价值）
  • 案例： 太阳马戏团剔除了动物表演（昂贵且有争议），增加了戏剧元素，创造了独特的艺术体验。

核心观点

  1. 可视化是沟通的基础： 商业模式太复杂，无法仅用文字描述。画布提供了一种通用的视觉语言，让不同背景的人（技术、市场、财务）能在同一个频道上对话。
  2. 商业模式是动态的： 画布不是静态的表格，而是动态的沙盘。你需要不断移动便利贴，测试不同的假设。
  3. 创新不仅仅是技术创新： 很多伟大的公司（如星巴克、戴尔）并没有发明新技术，而是发明了新的商业模式。
  4. 原型思维： 商业模式设计是一个迭代过程。先发散（设计多个备选方案），再收敛（选择最佳方案）。
  5. 以客户为中心： 一切设计都始于对客户痛点的深刻理解（移情图）。

金句摘录

“商业模式画布不仅仅是一个工具，它是一种思维方式。”

解析： 它强迫我们进行系统性思考，看到各个模块之间的逻辑联系，而不是孤立地看待产品或营销。

“不要爱上你的第一个想法。”

解析： 创业者最大的陷阱就是过早陷入对初步构想的自恋中。要像科学家一样，把商业模式看作一系列待验证的假设。

“失败得越快，成功得越早。”

解析： 在纸上（画布）失败的成本远低于在市场上失败。通过原型测试，尽早发现漏洞。

“伟大的商业模式能自我强化。”

解析： 优秀的模式中，各个模块是相互促进的。例如，Costco的低价吸引了会员，会员费补贴了低价，形成了飞轮效应。

应用指导

1. 创业者：从0到1设计商业模式

  • 场景： 你有一个创业点子，想梳理清楚。
  • 应用：
    • 画出画布： 找一面墙，贴上大白纸，画出9个格子。
    • 填空： 用便利贴把你的想法贴上去。先填CS（客户）和VP（价值），这是核心。
    • 检查逻辑： 你的VP真的能解决CS的痛点吗？你的RS（收入）能覆盖CS（成本）吗？
    • 寻找盲点： 看看哪个格子是空的或模糊的，那就是你的风险点。

2. 企业高管：战略转型与创新

  • 场景： 公司主营业务下滑，需要寻找第二曲线。
  • 应用：
    • 诊断现状： 画出当前的商业模式，分析哪里出了问题（是渠道老化？还是价值主张过时？）。
    • 设计未来： 组织跨部门团队，设计3-5个未来的商业模式原型。
    • 压力测试： 模拟竞争对手的反应，或者技术变革的影响，测试新模式的鲁棒性。

3. 个人发展：个人商业模式画布

  • 场景： 思考职业规划（参考《商业模式新生代（个人篇）》）。
  • 应用：
    • 客户细分： 谁是你的”客户”？（老板、同事、行业？）
    • 价值主张： 你能提供什么独特价值？（技能、资源、连接？）
    • 核心资源： 你拥有什么？（时间、知识、人脉、健康？）
    • 收入来源： 你获得什么回报？（工资、成就感、成长？）
    • 分析： 你的投入产出比合理吗？如何优化你的个人商业模式？
