《了不起的盖茨比》

弗朗西斯·斯科特·菲茨杰拉德（F. Scott Fitzgerald）

“于是我们奋力前行，却如同逆水行舟，注定要不停地被推回过去。“

作者介绍

菲茨杰拉德（F. Scott Fitzgerald，1896-1940）

生平背景

菲茨杰拉德，1896年9月24日出生于美国明尼苏达州圣保罗市。他是”爵士时代”的代言人和编年史家，与妻子泽尔达的奢靡生活成为那个时代的象征。44岁时因心脏病去世，生前并未获得应有的认可。

主要成就

  1. “爵士时代”的桂冠诗人：以文学定义了1920年代的美国
  2. 美国梦的批判者：揭示繁华表象下的空虚与幻灭
  3. 20世纪美国最伟大小说之一：《了不起的盖茨比》被列为”20世纪百大英文小说”第二名

主要内容

故事梗概

1922年夏天，叙述者尼克·卡拉威搬到纽约长岛，成为神秘富豪杰·盖茨比的邻居。盖茨比举办奢华派对，只为吸引住在对岸的旧情人黛西。

盖茨比出身贫寒，年轻时爱上富家女黛西，因贫穷被抛弃。他靠非法手段积累财富，只为赢回黛西。两人重逢后旧情复燃，但黛西开车撞死人后，盖茨比为她顶罪。黛西的丈夫汤姆指使死者丈夫杀死了盖茨比。

盖茨比的葬礼几乎无人参加，他追逐了一生的梦想被证明不过是幻影。

核心人物

杰·盖茨比

  • 白手起家的”美国梦”化身
  • 为爱不惜一切，最终幻灭

黛西·布坎南

  • 美丽而空洞
  • 代表盖茨比追逐的”绿灯”

核心观点

  1. 美国梦的幻灭：物质成功无法买到真正的幸福与爱情
  2. 过去不可追回：盖茨比想重复过去，注定失败
  3. 上流社会的腐朽：汤姆和黛西”把东西弄得乱七八糟，然后缩回他们的钱堆里”
  4. 纯真的代价：盖茨比的”了不起”在于他依然相信梦想

金句摘录

“于是我们奋力前行，却如同逆水行舟，注定要不停地被推回过去。”

解析：全书最后一句，概括了人类追逐梦想的悲剧性处境。

“盖茨比信奉那盏绿灯，信奉那个年年在我们面前渐渐远去的飘渺未来。”

解析：绿灯是盖茨比梦想的象征，永远可望不可即。
