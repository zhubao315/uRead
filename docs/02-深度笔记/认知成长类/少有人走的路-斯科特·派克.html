《少有人走的路》

斯科特·派克（M. Scott Peck）

“人生苦难重重。这是个伟大的真理，是世界上最伟大的真理之一。“

作者介绍

斯科特·派克（M. Scott Peck，1936-2005）

美国精神科医生、作家。《少有人走的路》出版于1978年，在《纽约时报》畅销书榜上停留超过600周，影响了数百万读者。

主要内容

第一部分：自律

四项自律原则

  1. 延迟满足

    • 先苦后甜，而非先甜后苦
    • 解决问题时先面对痛苦

  2. 承担责任

    • 为自己的问题负责
    • 不推卸、不逃避

  3. 尊重事实

    • 诚实面对现实
    • 不自欺欺人

  4. 保持平衡

    • 在各种需求间保持平衡
    • 学会放弃某些东西

第二部分：爱

真正的爱是什么

  • 爱是为了促进自己和他人心智成熟而付出努力
  • 爱不是感觉，而是行动

爱的误区

  • 坠入情网不是真爱（只是暂时的心理状态）
  • 依赖不是爱（是逃避成长的借口）

第三部分：成长与宗教

  • 心智成熟是终身的过程
  • 每个人需要建立自己的”宗教”（世界观）

核心观点

  1. 人生苦难重重：接受这个事实，痛苦反而减少
  2. 成长需要自律：自律是解决一切问题的基础
  3. 爱是行动而非感觉：真爱意味着付出努力帮助他人成长
  4. 心智成熟是终身任务：没有终点，只有过程

金句摘录

“人生苦难重重。这是个伟大的真理，是世界上最伟大的真理之一。”

解析：开篇名句，呼应佛教”苦谛”。接受这个事实，痛苦才能减轻。

“勇气，不意味着感受不到恐惧，而是尽管恐惧，依然行动。”

解析：勇气是带着恐惧前行，而非无畏。
