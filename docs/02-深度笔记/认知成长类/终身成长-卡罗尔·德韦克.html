cc 读书笔记

以下内容整合自 cc 仓库的深度阅读笔记。

《终身成长》（Mindset: The New Psychology of Success）

卡罗尔·德韦克（Carol Dweck）

“你认为你的能力是固定的，还是可以发展的？这个简单的信念，决定了你的一生。“

作者介绍

卡罗尔·德韦克（Carol Dweck）

生平背景 德韦克是斯坦福大学心理学教授，美国艺术与科学院院士。她是人格心理学、社会心理学和发展心理学领域的顶尖学者。

主要成就

  • 思维模式理论： 她用了几十年的时间研究”成功背后的心理机制”，提出了著名的”固定型思维”（Fixed Mindset）和”成长型思维”（Growth Mindset）理论。
  • 教育变革： 她的理论彻底改变了全球的教育理念，被比尔·盖茨极力推崇。

影响力 这本书（英文原名《Mindset》）是心理学领域的里程碑之作。它揭示了为什么同样聪明的人，有的能不断突破，有的却止步不前。它不仅适用于教育，也适用于商业、体育和人际关系。

创作背景

创作时间 本书于2006年出版。

社会环境 当时社会普遍崇拜”天赋”（Talent）。人们认为成功主要靠智商和天赋，努力只是辅助。这种观念导致了很多”伤仲永”的悲剧。

写作动机 德韦克在研究中发现，有些孩子面对困难时会兴奋（“我喜欢挑战！”），而有些孩子则会崩溃（“我不行，我笨”）。她想搞清楚这种差异的根源，并告诉大家：思维模式是可以改变的。

大纲

第一部分：两种思维模式

  • 固定型思维 vs 成长型思维。
  • 思维模式如何影响我们的生活。

第二部分：思维模式的真相

  • 关于能力。
  • 关于努力。
  • 关于失败。

第三部分：应用领域

  • 体育：冠军的思维。
  • 商业：领导力的思维。
  • 人际关系：恋爱的思维。
  • 教育：父母与老师的思维。

第四部分：如何改变

  • 通往成长型思维的旅程。

主要内容

1. 两种思维模式

  • 固定型思维（Fixed Mindset）： 认为智力、才能、性格是天生固定不变的。
    • 目标： 证明自己（“看我多聪明”）。
    • 看待努力： 努力是无能的表现（“天才不需要努力”）。
    • 看待失败： 失败定义了我的身份（“我失败了” = “我是个失败者”）。
    • 看待挑战： 逃避（怕暴露缺点）。
  • 成长型思维（Growth Mindset）： 认为智力、才能、性格是可以通过努力和学习不断发展的。
    • 目标： 提升自己（“我想学到东西”）。
    • 看待努力： 努力是通往精通的必经之路。
    • 看待失败： 失败是反馈，是学习的机会（“我还没学会”）。
    • 看待挑战： 拥抱（因为能让我变强）。

2. 脑科学基础

  • 神经可塑性： 大脑像肌肉一样，越练越强。当我们学习新东西、克服困难时，神经元之间的连接会变得更紧密。这为成长型思维提供了科学依据。

3. 努力的意义

  • 固定型： 认为”如果我需要努力，说明我不够聪明”。
  • 成长型： 认为”即使是天才，也需要通过努力才能发挥潜能”。乔丹、莫扎特都是通过刻苦练习才成为大师的。

4. 虚假的成长型思维

  • 误区： 以为”成长型思维”就是”保持乐观”或”表扬努力”。
  • 真相： 真正的成长型思维是关注过程，关注策略，关注从失败中学习。如果孩子考砸了，你只说”没关系，你很努力了”，这是无效的安慰。你应该问：“我们能从这次错误中学到什么？下次用什么新方法？“

5. 改变的四步法

  1. 接受： 承认自己心里有一个”固定型思维的小人”（每个人都有）。
  2. 观察： 观察这个小人什么时候会出现（比如遇到挫折时，它会说”你不行”）。
  3. 命名： 给这个小人起个名字（比如”丧气鬼”）。
  4. 教育： 用成长型思维去教育它，带它一起走（“丧气鬼，我知道你怕失败，但我们要试一试，失败了也没关系”）。

核心观点

  1. 能力是流动的： 现在的能力不代表未来的能力。只要方法得当，任何人都能在任何领域取得进步。
  2. 失败不是终点： 在成长型思维者眼里，没有”失败”（Failure），只有”还没成功”（Not Yet）。
  3. 表扬的陷阱： 不要表扬孩子的智力（“你真聪明”），这会培养固定型思维（为了保持”聪明”的标签而不敢冒险）。要表扬孩子的过程（“你刚才尝试了多种方法，真棒”），这会培养成长型思维。
  4. 性格也可以改变： 害羞、暴躁都不是天生的定局，都可以通过练习来改变。

金句摘录

“你认为你的能力是固定的，还是可以发展的？”

解析： 灵魂之问。

“在固定型思维的世界里，失败是关于你的定义。在成长型思维的世界里，失败是关于你的行动。”

解析： 失败观的区别。

“努力是让能力变现的唯一途径。”

解析： 天赋+努力=成就。

“Not Yet（尚未）。”

解析： 最有力量的一个词。

应用指导

1. 正确表扬孩子

  • 场景： 孩子考了100分。
  • 应用：
    • 不要说： “哇，你真聪明！天才！“（这让他以后不敢考99分）。
    • 要说： “哇，我看到你复习得很认真，还把错题都搞懂了，你的努力有了回报！“（强化过程）。

2. 面对职场挫折

  • 场景： 方案被老板批了一顿。
  • 应用：
    • 固定型反应： “老板针对我”、“我不适合干这个”（自我攻击或攻击他人）。
    • 成长型反应： “老板指出的问题确实存在。虽然被骂很难受，但这正是我提升方案质量的机会。我要去请教一下怎么改。“

3. 学习新技能

  • 场景： 学编程觉得太难，想放弃。
  • 应用：
    • 告诉自己：“觉得难是因为我的大脑正在建立新的神经连接。这种’难受’的感觉，就是’变强’的感觉。”
    • 把”我不行”改成”我还不行”。
