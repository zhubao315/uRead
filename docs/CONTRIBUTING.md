# Contributing to uRead

## 贡献方式

- 新增一本书的深度笔记
- 优化已有笔记的结构、论证或表达
- 补充知识卡片与双向链接
- 完善文档、脚本和 Agent 集成能力

## 内容规范

- 所有内容使用 Markdown 编写
- 新笔记基于 `content/templates/deep-reading-template.md`
- 每篇内容必须包含 frontmatter
- 标题、标签、摘要必须可读且可检索
- 优先增加双向链接，避免孤立内容

## Pull Request 要求

- PR 标题明确说明改动范围
- 说明为什么改、改了什么、如何验证
- 涉及新增书籍内容时，补充摘要与标签
- 涉及脚本修改时，说明生成产物的变化

## 审核标准

- 内容结构是否完整
- 论证是否清晰
- 是否具备复用价值
- 是否符合 Agent 检索友好要求
